# Tic Tac Toe

This is a simple implementation of Tic-Tac-Toe except it uses WebRTC for
peer-to-peer communication. The game is played in the browser and uses WebRTC to
connect two players directly without a server.

Play here: https://tictactoe.purduehackers.com

This is the complete code for the workshop run on 04/01/2025 on Peer-to-Peer Video
Games by Purdue Hackers! You can find the slides for the workshop [here]().
